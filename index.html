Power BI Excel ETL DAX Storytelling

Power BI | Varejo de Bebidas

Um estudo de caso sobre como transformar dados de vendas complexos em um centro de comando para o varejo. Através de storytelling interativo, análise de rentabilidade e um design centrado no usuário (com temas light/dark), este projeto prova que a inteligência de dados bem aplicada é o motor do crescimento e da lucratividade do negócio.

A estratégia foi armar o gestor com uma ferramenta de descoberta, transformando a análise de um relatório passivo em uma investigação ativa. O painel foi arquitetado não para exibir KPIs estáticos, mas para criar um diálogo com os dados, permitindo identificar a performance de cada produto, otimizar o mix de vendas e responder a perguntas de negócio em tempo real. Trata-se de converter o analista de um observador em um estrategista proativo.

Power BI Excel Storytelling

Power BI | Finanças Pessoais

Um estudo de caso sobre como transformar dados financeiros caóticos em um painel de controle para a vida. Usando storytelling visual, filtros dinâmicos e tooltips inteligentes, este projeto demonstra como a clareza impulsiona o controle e a otimização de recursos. É a prova de que dados bem apresentados são o alicerce da decisão inteligente.

A estratégia foi capacitar o usuário a conduzir sua própria investigação, transformando a análise de passiva para ativa. O painel foi arquitetado não para exibir números, mas para responder perguntas, revelando os padrões que permitem o planejamento e a prosperidade.

HTML CSS

Portfolio | JR Cronogramas

A construção de um hub digital de conversão para a marca JR Cronogramas. Este projeto eleva o "link na bio" a um ativo estratégico, projetado para transformar seguidores do Instagram em clientes e aumentar a credibilidade da marca.

O princípio foi mover a marca de um "terreno alugado" (Instagram) para um "terreno próprio". Ao centralizar a jornada do cliente em uma plataforma controlada, criamos um funil de vendas mensurável e blindamos o negócio contra a volatilidade dos algoritmos.
